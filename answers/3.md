Есть две функции:

isNaN() - функция, которая определяет, является ли передаваемое значение NaN и возвращает логическое значение. Функция возвращает true, если передаваемое значение NaN и его тип Number, в противном случае возвращаемое значение false.

isFinite() - функция, которая определяет, является ли переданное значение конечным числом и возвращает логическое значение. Метод возвращает true, если передаваемое значение конечное число, в противном случае возвращаемое значение false.
